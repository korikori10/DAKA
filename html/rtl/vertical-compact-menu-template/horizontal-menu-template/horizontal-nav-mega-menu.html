The Horizontal navigation mega menu usage & customization option.

Description

A mega menu is typically defined as a drop down interface that is triggered by the user hovering/clicking over a link or defined area. This dropdown usually shows all options in one main, mega-panel and oftentimes groups related topics into categories.

Mega menu shows all the options in one large panel, it contains any type of components like form inputs, tabs, buttons, accordion, table & card.

Robust Admin mega menu provides best usability and navigation options, it is also easy to use and configure.

HTML Markup

This section contains HTML Markup to create Mega Menu with different options.

Flyout Menu Classes

  • Line no 2: Wrap the dropdown’s trigger and the dropdown menu within .dropdown.
  • Line no 3: To toggle menu use the .dropdown-toggle.
  • Line no 4: .dropdown-menu class wrap the dropdown menu.
  • Line no 5: .dropdown-submenu sub menu class.
  • Line no 7: .dropdown-item menu item class used for menu list.
          
            <!-- Flyout Menu -->
            <li class="dropdown nav-item">
              <a href="#" data-toggle="dropdown" class="dropdown-toggle nav-link"><i class="icon-layers"></i><span>Flyout Menu</span></a>
                <ul class="dropdown-menu">
                  <li data-menu="dropdown-submenu" class="dropdown dropdown-submenu"><a href="#" data-toggle="dropdown" class="dropdown-item dropdown-toggle" aria-expanded="false">First level with child</a>
                    <ul class="dropdown-menu">
                      <li><a href="#" data-toggle="dropdown" class="dropdown-item">Second level</a>
                      </li>
                      <li data-menu="dropdown-submenu" class="dropdown dropdown-submenu"><a href="#" data-toggle="dropdown" class="dropdown-item dropdown-toggle">Second level with child</a>
                        <ul class="dropdown-menu">
                          <li><a href="#" data-toggle="dropdown" class="dropdown-item">Third level</a>
                          </li>
                        </ul>
                      </li>
                    </ul>
                  </li>
                </ul>
              </li>
              <!--/ Flyout Menu -->

Mega Menu Classes:

  • Line no 2: .mega-dropdown menu parent class used in the <li> tag.
  • Line no 4:#mega-dropdown-menu menu content class used in the <ul> tag.
  • Line no 9:.mega-menu-sub sub menu class used in the <ul> tag.

Fixed Width Menu Classes:

Fixed menu use the same bootstrap dropdown menu with fixed width class.

  • Line no 5: Has fixed menu class width-700, you can use any fixed width class as per requirement. Check robust template helpers classes for more information.
  • Line no 7: col-md-* Davide the fixed width menu in to columns using bootstrap grid layout.

CSS Classes

This table contains all classes which can be used in the mega menu. You can combine them as per your template requirements.

All these options can be set via following classes:

Classes Description
.mega-dropdown The wrapper class for the mega menu, it will set mega menu position to static to set the mega menu position.
#mega-dropdown-menu

This is the mega menu columns wrapper class. It set mega menu width to 100%.
.mega-menu-sub

Inside mega menu columns, .mega-menu-sub class required to wrap all .dropdown-item
.width-*

To set the fixed with menu width, .width-* class required. EX: To set width 300 use class .width-300

JADE Configuration

Robust Admin use JADE as template engine to generate pages and whole template quickly using node js, for getting start with JADE usage & template generating process please refer template documentation.

JADE Variables

This table contains required JADE variables to generate horizontal navigation layout.

Object Description
megaMenuCols This object content number of columns used in mega menu with all content.
megaMenu This object used in megaMenuCols, it content mega menu all level menu items.
submenu This object used to define megaMenu object sub menu items.
