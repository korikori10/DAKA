Green Palette Classes

Green color palette contains 13 colors: 1 main green color and 4 lighten colors, 4 darken colors and 4 Accent colors. All colors works perfectly with any bootstrap components, form components or any robust controls, that makes Robust very effective, flexible and configurable.

Please note: You can use .bg-green and .btn-green as main colors, but if you want to use lighten, darken or accent colors, use bg-* bg-lighten-NUMBER for lighten color, bg-* bg-darken-NUMBER for darken color and bg-* bg-accent-NUMBER for accent colors.

.bg-green Green Class for background color.
