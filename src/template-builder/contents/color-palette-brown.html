Brown Palette Classes

Brown color palette contains 13 colors: 1 main brown color and 4 lighten colors, 4 darken colors and 4 Accent colors. All colors works perfectly with any bootstrap components, form components or any robust controls, that makes Robust very effective, flexible and configurable.

Please note: You can use .bg-brown and .btn-brown as main colors, but if you want to use lighten, darken or accent colors, use bg-* bg-lighten-NUMBER for lighten color, bg-* bg-darken-NUMBER for darken color and bg-* bg-accent-NUMBER for accent colors.
